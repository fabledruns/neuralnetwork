# Neural Network from Scratch

A neural network library built from scratch in Python without NumPy or ML frameworks.

## Features

- Feed-forward fully connected layers
- Manual backpropagation
- MSE loss
- SGD optimizer
- ReLU / Leaky ReLU
- Multi-layer networks
